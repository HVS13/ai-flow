# Presentation Skill

`core/skills/ppt.md`

Use this skill for slide decks, pitch decks, briefings, and visual storylines.

## Planner mode

Planner should:

- define audience
- define goal
- define message flow
- decide slide count
- decide tone and level of detail
- define required slide types

## Builder mode

Builder should:

- create slide outline
- organize storyline
- draft slide text and speaker notes
- keep slide count within scope
- avoid adding new strategy or scope creep
- write a completion report

## Recommended outputs

- slide outline
- slide content draft
- speaker notes
- optional appendix content

## Workspace

Use:

```text
workspaces/ppt/<ticket-id>/
```

## Verification

- audience and goal are addressed
- slide sequence is logical
- message is concise
- each slide has one main point
- supporting evidence is included where required
- file format matches ticket request

## Completion report should include

- summary
- files created or changed
- storyline kept
- verification steps
- missing data or weak points
- suggested follow-up tickets
- confidence
