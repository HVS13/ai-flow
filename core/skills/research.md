# Research Skill

`core/skills/research.md`

Use this skill for analysis, comparisons, decision support, literature review, evidence gathering, and synthesis.

## Planner mode

Planner should:

- define the research question
- define scope boundaries
- define required deliverables
- define evidence quality expectations
- define what is out of scope

## Builder mode

Builder should:

- gather relevant evidence
- compare sources
- separate facts from assumptions
- summarize findings
- write a completion report

## Recommended outputs

- research notes
- comparison table
- recommendation memo
- executive summary

## Workspace

Use:

```text
workspaces/research/<ticket-id>/
```

## Verification

- the research question is answered directly
- sources or assumptions are stated
- limitations are clear
- recommendations are actionable
- the output stays inside the ticket scope

## Completion report should include

- summary
- files created or changed
- evidence used
- assumptions
- limitations
- suggested follow-up tickets
- confidence
